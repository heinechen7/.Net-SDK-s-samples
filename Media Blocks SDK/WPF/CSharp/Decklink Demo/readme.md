# Media Blocks SDK .Net - Decklink Demo (WPF)

Decklink Demo is an application that uses the Media Blocks SDK .Net to capture video from Blackmagic Decklink devices.

## Supported frameworks

- .Net 4.7.2
- .Net Core 3.1
- .Net 5
- .Net 6
- .Net 7
